# BPC CMS Frontend

React + TypeScript + Vite frontend for BPC CMS.

## Scripts

- `npm run dev`: Start the development server.
- `npm run lint`: Run ESLint.
- `npm run build`: Type-check and build for production.
- `npm run preview`: Preview the production build locally.

## Runtime Requirements

- Node.js `24.x`
- npm `11.6.2`

Install and run commands are blocked automatically if these versions do not match.

## Tech Stack

- React 19
- TypeScript 5
- Vite
- ESLint
- Tailwind CSS + shadcn/ui
- Axios
- Zod
- Zustand

## Environment

Create `frontend/.env`:

```env
VITE_API_URL=http://localhost:3000/api/v1
VITE_API_BASE_URL=http://localhost:3000
VITE_APP_NAME=Bhola Programming Club
```
